# Float

  • 요소를 둥둥 떠다니게 만든다.
  • float: left - 왼쪽에 떠다니게 만든다.
  • float: right - 오른쪽에 떠다니게 만든다.
  • float: none - float설정을 제거한다.(기본값)

# clear

  • clear: none - 모든 float 요소를 허락한다.(기본설정)
  • clear: left - 왼쪽의 float 요소를 허락하지 않는다.
  • clear: right - 오른쪽의 float 요소를 허락하지 않는다.
  • clear: both - 양쪽의 float 요소를 허락하지 않는다.
