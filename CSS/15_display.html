# Display

  • 해당 요소가 어떻게 보여야 하는지를 결정하는 속성(property)
  • 대부분의 요소들은 display 속성의 기본값으로 inline과 block을 가지고 있다

# display: block

항상 새 줄에서 시작하고, 너비를 100% 차지하려고 한다.

p content

div content

heading content

heading content

heading content

heading content

heading content
heading content

# display: inline

새 줄에서 시작하지 않고, 필요한 만큼만의 너비를 차지한다(글자취급)

span content bold content strong content link content

# display: none

해당 요소를 보이지 않게 숨긴다.(영역도 사라진다.)

# visiblty: hidden

해당 요소를 보이지 않게 만든다.(영역은 그대로 유지된다.)
