# 내용이 넘칠 때 처리

overflow: visible
- 넘치는 내용을 그냥 보여준다. (기본 설정)
overflow: hidden
넘치는 내용을 숨긴다
overflow: scroll
- 내용이 넘치면 스크롤을 생성한다 (필요없는 스크롤까지 생성)
overflow: auto
- 필요한 곳에만 스크롤을 생성한다
※ overflow-x와 overflow-y를 통해 각각 다르게 설정할 수 있다
