# 크기조절 단위들

  1. 절대 길이 단위들
    • px: 픽셀 (화소)
    • mm: 밀리미터
    • cm: 센치미터
    • in: 인치
  2. 상대 길이 단위들
    • em: 현재 요소의 font-size에 기반한 크기 조절
      (ex: 2.0em일 때 font-size의 2배 크기가 된다.)
    • vw: viewport width, 웹 페이지에 접속한 브라우저의 너비에 기반한 크기 조절
      (ex: 1vw일 때 현재 웹 브라우저 너비의 1%를 차지한다.)
    • vh: viewport height, 웹 페이지에 접속한 브라우저의 높이에 기반한 크기 조절
      (ex: 1vh일 때 현재 웹 브라우저 높이의 1%를 차지한다.)
