# display: flex

  • 상황에 따라 내부 요소들의크기를 자동으로 조절하는 형태의 레이아웃을 설정한다
  • display: flex를 설정한 요소가 컨테이너가 된다
  • 자식 요소들은 모두 flex의 item들로 활용된다
  • flex container에서 컨트롤하는 방법과 flex item에서 컨트롤하는 방법이 있다
