# 가상 클래스(Pseudo Class)

    P태그의 내용

  • 가상 클래스를 이용해 해당 요소의 특징 상황을 선택할 수 있다.
  • 선택자 뒤에 :을 붙여서 원하는 상황을 선택한다.

# 가상 클래스 종류

    first-child, nth-child(n/even/odd), last-child
    - 요소가 부모의 특정 번째 요소일 때를 선택한다.
    first-of-type, nth-of-type(n/even/odd), last-of-type
    - 요소가 같은 타입들 중에서 부모의 특정 번째 요소일 때를 선택한다.
    hover
    - 해당 요소에 마우스가 올라가 있는 상태를 선택한다.
    link
    - 해당 요소가 일반적인 상태의 링크인 상태를 선택한다.
    active
    - 해당 요소가 활성화된 링크인 상태를 선택한다.
    visited
    - 해당 요소가 방문했던 링크인 상태를 선택한다.
