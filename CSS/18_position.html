# 요소들의 포지션 설정하기

  • position property를 통해 요소의 위치를 마음대로 설정할 수 있다.
  • position property는 포지션이 설정된 가장 가까운 부모를 기준으로 동작한다.
  • position이 static 이외의 것으로 설정된 요소는 top, right, left, bottom property를 설정 할 수 있다.
  • 포지션이 설정된 요소 끼리는 겹칠 수 있는데, 겹친 요소들 중 누가 위로 올라 올지는 z-index property가 결정한다.
    ※ 아무리 z-index가 높더라도 부모보다 밑에 깔릴 수는 없다.
  • static: 기본값, 포지션이 설정되지 않은 상태.
  • relative: 요소가 원래 있던 위치에서 상대적인 위치를 설정한다.
  • fixed: 웹 브라우저에서 항상 고정적인 위치를 차지한다.
  • absolute: 절대 위치를 설정한다.
  • sticky: 스크롤이 내려갈때 따라 내려오도록 설정한다. 원래 있던 위치에서 위치 설정
