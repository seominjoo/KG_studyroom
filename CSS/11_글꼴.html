# font-family propert로 글꼴 설정하기

  • font-family 속성으로 원하는 글꼴을 선택할 수 있다.
  • ,를 찍어서 하나의 요소에 여러 글꼴을 설정해 둘 수 있다.
  • 원하는 글꼴이 유효하지 않은 경우 (설치되어 있지 않은 경우, 인터넷 연결 불안정) 뒤 쪽에 설정된 다음 글꼴을 적용한다.
  • 사용자의 컴퓨터에 해당 글꼴이 불러지지 않는 경우에 대비해 여러개의 글꼴을 설정 해놓는 것이 좋다.

# font-weight: 글자 굵기 조절

  • font-weight:200
  • font-weight:300
  • font-weight:400
  • font-weight:500
