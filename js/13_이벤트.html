# 이벤트 처리

  1. HTML속성을 이용하는 방식 (속성 이름은 이벤트명 앞에 on을 붙인다)
  2. 자바스크립트를 이용해 속성에 추가하는 방식
  3. 해당 노드에 이벤트 리스너(콜백 함수)를 추가하는 방식

# 엄청 많은 이벤트들이 존재한다.

  • click : 마우스 클릭
  • mouseover : 마우스가 해당 요소위에 올라감
  • mouseout : 마우스가 해당 요소를 벗어남
  • mousedown : 마우스 버튼이 눌림
  • mouseup : 마우스 버튼이 원래대로 돌아옴
  • keydown : 키보드 키가 눌림
  • keypress : 키보드가 오래 눌림
  • keyup : 키보드 버튼이 원래대로 돌아옴
  • load : 페이지의 로딩이 끝남
  • focus : 커서가 해당 요소에 있을 때
  • focusout : 포커스를 잃을 때
  • submit : submit이 발생했을 때
  • https://developer.mozilla.org/en-US/docs/Web/Events
